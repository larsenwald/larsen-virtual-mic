Larsenwald Virtual Mic // setup required
VB-Audio Virtual Cable is required
Larsenwald Virtual Mic routes your microphone through a DSP plugin chain and outputs to a virtual mic device that other apps can use.

To do this, it needs to install VB-Audio Virtual Cable — a free, widely-used audio driver — on your system.

The driver will be renamed to Larsenwald VM (Don't Touch) and Larsenwald (Virtual Mic) and configured exclusively for this app.
Larsenwald Virtual Mic // driver conflict
VB-Audio Virtual Cable is already installed
Larsenwald Virtual Mic found an existing VB-Audio Virtual Cable installation on your system.

⚠ This app will take full ownership of that driver. It will be uninstalled and reinstalled fresh, then renamed and reconfigured exclusively for Larsenwald Virtual Mic.

If you're currently using VB-Audio for anything else — another app, your own routing setup — that configuration will be permanently overwritten.

If that's not okay, do not continue.
